# CreatedbyJohnny - Portfolio

A modern portfolio website for a graphic and web designer, built with React, TypeScript, and Vite.

## Getting Started

**Prerequisites:**
- Node.js or Bun installed

**Installation:**
```bash
bun install
```

**Development:**
```bash
bun run dev
```

**Build:**
```bash
bun run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui components

## Project Structure

- `src/components/` - React components
- `src/pages/` - Page components
- `src/assets/` - Images and media
- `src/lib/` - Utility functions

## Deployment

This project is deployed to GitHub Pages. Push to the `main` branch to trigger automatic deployment.

Live at: https://soartzs.github.io/CreatedbyJohnny/
